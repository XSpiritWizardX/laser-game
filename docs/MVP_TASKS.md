# MVP Tasks

1. Ship movement and firing controls (WASD/Arrows, Space).
2. Enemy spawning with increasing difficulty.
3. Collision detection for lasers, enemies, and the player.
4. Score, lives, and level HUD.
5. Start, pause, and game-over overlays.
6. Visual feedback for hits (sparks, flashes, screen shake).
7. Basic responsive layout for desktop and mobile screens.
8. Touch controls for mobile (virtual stick + fire button).
9. Sound effects and a mute toggle.
10. Render deployment configuration and publish instructions.
